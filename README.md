✔ 프로젝트 기간 : 19년 09월 02일 ~ 12월 15일 (총 4주)

✔ 개발 목적 : 학과 사무실에 가지 않고 강의실 이용에 대한 접근성을 높이기 위한 캡스톤 프로젝트
